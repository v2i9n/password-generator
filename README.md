# 🔐 password-generator - مولد كلمات السر القوية

مشروع بايثون يولد لك كلمات سر معقدة ومستحيلة الاختراق. من برمجة **Hussein** 👨‍💻

### ✨ المميزات:
- تختار طول كلمة السر اللي تبيها بنفسك
- يخلط حروف كبيرة + صغيرة + أرقام + رموز `!@#$%^&*`
- يمنعك من كلمات السر الضعيفة أقل من 4 خانات
- تقدر تولد باسوردات لا نهائية لين تطلع من البرنامج

### 🚀 كيف تشغله:
1. نزّل ملف `password_generator.py`
2. شغله بالأمر: `python password_generator.py`
3. اكتب الطول اللي تبيه واضغط Enter
